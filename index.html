🎯 项目简介

AutoStockWeb 是一个基于 Python 的 A 股实时数据获取工具，专注于机器人概念股的数据采集与分析。通过 akshare 接口获取东方财富实时行情，支持自定义股票池扩展。

⚡ 实时数据

通过 akshare 获取 A 股实时行情，包含价格、涨跌幅、成交量等关键指标

🤖 机器人概念

内置 10 支核心机器人概念股，覆盖埃斯顿、汇川技术、绿的谐波等龙头

📊 数据持久化

自动保存 JSON 格式数据，支持历史数据查看与回溯分析

🔧 易于扩展

模块化设计，通过配置文件即可扩展股票池，支持全市场数据获取

🚀 快速开始

1. 安装依赖

cd autoStockWeb
pip install -r requirements.txt

2. 获取实时数据

python main.py

3. 查看历史数据

python main.py --list
python main.py --read stock_data_20260523_023237.json

4. 调试模式

python main.py --debug
