## 4. 使用说明

### 4.1 安装依赖

```bash
cd E:\ccProject\autoStockWeb
pip install -r requirements.txt
```

依赖列表：
- akshare >= 1.15.0（A股数据接口）
- pandas >= 2.0.0（数据处理）
- requests >= 2.31.0（HTTP请求）

### 4.2 获取实时数据

```bash
python main.py
```

输出示例：
```
股票数据摘要:
------------------------------------------------------------
名称         代码       当前价        涨跌幅
------------------------------------------------------------
埃斯顿        002747   26.11      9.98%
汇川技术       300124   75.00      -0.73%
机器人        300024   16.09      7.70%
...
------------------------------------------------------------
```

数据保存至：`data/stock_data_YYYYMMDD_HHMMSS.json`

### 4.3 查看历史数据文件

```bash
python main.py --list
```

### 4.4 读取指定数据文件

```bash
python main.py --read stock_data_20260523_023237.json
```

### 4.5 调试模式

```bash
python main.py --debug
```

## 5. 扩展全部股票

编辑 `config/stocks.json`，添加更多股票：

```json
{
  "name": "扩展股票池",
  "stocks": [
    {"code": "000001", "name": "平安银行", "market": "SZ"},
    {"code": "600000", "name": "浦发银行", "market": "SH"}
    // ... 更多股票
  ]
}
```

**注意**：
- `market` 字段：`SH` = 上海，`SZ` = 深圳
- 代码不需要加后缀（如 `sh688017`），程序会自动格式化

## 8. 注意事项

1. **A股交易时间**：工作日 9:30-11:30, 13:00-15:00，非交易时间获取的数据可能为收盘价或空值
2. **请求频率**：程序已内置 0.5 秒间隔，避免请求过快被封
3. **数据保存**：JSON 文件保存在 `data/` 目录，已加入 `.gitignore` 不提交到 Git
4. **错误处理**：单支股票失败不影响其他股票，会记录错误日志
